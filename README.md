# Quorum

The fastest way to make better business decisions.
A private network of founders who tell you the truth.

## Local preview

Open `index.html` in a browser, or run any static server:

```
npx http-server . -p 5173
```

Then visit http://localhost:5173

## Deploy

This is a static site. `vercel.json` is preconfigured with security headers and clean URLs.

**Vercel (recommended):**
1. Go to https://vercel.com/new
2. Import the GitHub repo
3. Click Deploy — no settings to change

Every push to `main` auto-deploys.

## Waitlist

Email submissions are sent to a Formspree endpoint configured in `index.html`.
Replace the URL there if you switch providers.
